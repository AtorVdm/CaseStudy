FactoryObject

This managed object class is used to hold the factory classes for other managed objects and to instantiate new managed objects. This is the only way managed object types get into the system and the only way instances of those types are created (unless they are being restored).

Factory Messages
OperationDescription
create: adaptorClass creates a managed object that acts as a factory for creating instances of the underlying managed object

Operational Messages
OperationReturnDescription
Wild Card P2Object all messages are sent to the Managed Object being created. See the object's factory documentation for commands accepted.
