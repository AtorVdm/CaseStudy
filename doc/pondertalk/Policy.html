Policy

This class implements policies within the SMC. It is the base class for more specialised policy classes and manages all the common work.

Factory Messages
OperationDescription

Operational Messages
OperationReturnDescription
active: aBoolean P2Object makes the policy active if aBoolean is true else makes the policy inactive. Returns aBoolean.
attach: aManagedObject self attaches this policy to aManagedObject. This policy may be attached to more than one managed object. This command forms part of the proximity event bus.
